# SPA Template - MeteorJS + Materialize UI 


# Instalation

## Linux / OSX
```
curl https://install.meteor.com/ | sh
meteor npm install
```
## Windows
```
https://install.meteor.com/windows
meteor npm install
```


# App start

Production version
```
meteor --production
```
Developer version
```
meteor
```
